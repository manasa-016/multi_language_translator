# 🌍 Multi-Language Translator Web App

A simple and powerful Flask-based web application that translates text into multiple languages instantly.

🔗 Live Demo: https://multi-language-translator-1-7jpl.onrender.com/

## 🚀 Features

- 🌐 Translate text into multiple global languages

- ⚡ Real-time translation

- 🎯 Clean and responsive user interface

- 📱 Mobile-friendly design

- 🔁 Easy language selection

- 🧠 Backend powered by Python

## 🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript

- Backend: Python (Flask)

- Deployment: Render


## ⚙️ How to Run Locally

1️⃣ Clone the repository
git clone https://github.com/your-username/multi-language-translator.git
cd multi-language-translator

2️⃣ Create virtual environment (recommended)
python -m venv venv
venv\Scripts\activate   # Windows

3️⃣ Install dependencies
pip install -r requirements.txt

4️⃣ Run the application
python app.py

5️⃣ Open in browser
http://127.0.0.1:5000/


## 🧠 How It Works

- User enters text in the input box.

- Selects a target language.

- Flask backend processes the request.

- Translation API/library converts the text.

- Translated result is displayed instantly on the webpage.

## 📦 Dependencies

Check requirements.txt for required libraries.
Common libraries may include:

- Flask

- googletrans / deep-translator (or whichever you used)
